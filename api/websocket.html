WebSocket Chat

Real-time streaming chat powered by Claude. Connect via WebSocket, send messages, and receive streamed AI responses with conversation history support.

WebSocket URL wss://p417pa2mu2.execute-api.us-east-1.amazonaws.com/prod?token=<idToken>
Authentication is done via the token query parameter when connecting. Use the idToken from the login response. The WebSocket authorizer validates the token and extracts the user ID.

Connection Flow

The WebSocket API uses a simple message-based protocol. After connecting, send JSON messages and receive streamed responses.

Message Actions
ActionDescription
sendMessageSend a chat message and receive a streamed AI response.
load_historyLoad paginated conversation history.
clear_conversationDelete all conversation history for the user.
sendMessage Parameters
FieldTypeDescription
actionstring requiredMust be "sendMessage".
questionstring requiredThe message/question to send.
model_idstring optional"basic" (Claude Haiku 4.5) or "premium" (Claude Sonnet 4). Default: "basic".
system_promptstring optionalCustom system prompt for the AI.
Response Events (streamed)
TypeDescription
startProcessing has begun. { "type": "start", "message": "Processing..." }
chunkA text chunk of the response. { "type": "chunk", "text": "..." }
endResponse complete. { "type": "end", "message": "Response complete" }
load_history Parameters
FieldTypeDescription
actionstringMust be "load_history".
startnumber optionalOffset start. Default: 0.
endnumber optionalOffset end. Default: 10.
 
// Send a chat message
{
  "action": "sendMessage",
  "question": "Explain quantum computing",
  "model_id": "basic",
  "system_prompt": "You are a helpful assistant"
}
// Load conversation history
{
  "action": "load_history",
  "start": 0,
  "end": 20
}
// Clear all conversation history
{
  "action": "clear_conversation"
}
// Stream events received:

// 1. Processing started
{ "type": "start", "message": "Processing your request..." }

// 2. Text chunks (multiple)
{ "type": "chunk", "text": "Quantum computing is" }
{ "type": "chunk", "text": " a type of computation" }

// 3. Response complete
{ "type": "end", "message": "Response complete" }

JavaScript Example

// Connect to WebSocket
const token = 'your-id-token';
const ws = new WebSocket(
  `wss://p417pa2mu2.execute-api.us-east-1.amazonaws.com/prod?token=${token}`
);

ws.onopen = () => {
  console.log('Connected');

  // Send a message
  ws.send(JSON.stringify({
    action: 'sendMessage',
    question: 'What is machine learning?',
    model_id: 'basic'
  }));
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);

  if (data.type === 'chunk') {
    process.stdout.write(data.text); // Stream text
  } else if (data.type === 'end') {
    console.log('\nDone!');
  }
};
import websocket
import json

token = "your-id-token"
url = f"wss://p417pa2mu2.execute-api.us-east-1.amazonaws.com/prod?token={token}"

ws = websocket.create_connection(url)

# Send a message
ws.send(json.dumps({
    "action": "sendMessage",
    "question": "What is machine learning?",
    "model_id": "basic"
}))

# Receive streamed response
while True:
    result = json.loads(ws.recv())
    if result["type"] == "chunk":
        print(result["text"], end="")
    elif result["type"] == "end":
        print("\nDone!")
        break

ws.close()
