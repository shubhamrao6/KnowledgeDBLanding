RAG Search

Ask natural language questions and receive AI-generated answers grounded in your uploaded documents. Powered by Amazon Bedrock Knowledge Base retrieval and Claude for answer generation.

Requires authentication. Include Authorization: Bearer <idToken> and x-api-key: <apiKey> in your request headers. Both are returned from the login response.
The search pipeline: your question → vector similarity search in Bedrock KB (filtered by user + knowledge base) → relevant document chunks retrieved → Claude generates a grounded answer with source citations.
POST

Search

/search

Submit a natural language question scoped to a specific knowledge base. The API retrieves relevant document chunks using vector search, then generates an AI answer grounded in those chunks.

Request Body
ParameterTypeDescription
questionstring requiredNatural language question to answer.
knowledge_dbstring requiredKnowledgeDB ID to search within.
top_knumber optionalNumber of document chunks to retrieve. Default: 3, max: 10.
Response Fields
FieldTypeDescription
answerstringAI-generated answer based on retrieved documents.
sourcesarrayArray of source objects with document references.
questionstringOriginal question echoed back.
knowledgeDbstringKnowledgeDB ID searched.
Source Object
FieldTypeDescription
doc_idstringDocument ID of the source.
titlestringFilename of the source document.
chunkstringRelevant text excerpt (max 500 chars).
scorenumberRelevance score from vector search.
 
// POST /search
// Header: Authorization: Bearer <idToken>
{
  "question": "What are the key benefits of microservices?",
  "knowledge_db": "281e85f4-00b8-4613-b518-a3b4679e8d4c",
  "top_k": 3
}
// 200 OK
{
  "answer": "Based on the documents, the key benefits of microservices include independent deployment, technology flexibility, and improved fault isolation...",
  "sources": [
    {
      "doc_id": "a1b2c3d4-...",
      "title": "microservices-guide.md",
      "chunk": "Microservices architecture enables teams to deploy services independently...",
      "score": 0.89
    }
  ],
  "question": "What are the key benefits of microservices?",
  "knowledgeDb": "281e85f4-00b8-4613-b518-a3b4679e8d4c"
}
# cURL example
curl -X POST \
  https://fgbpqt2pq6.execute-api.us-east-1.amazonaws.com/prod/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <idToken>" \
  -d '{
    "question": "What are the key benefits?",
    "knowledge_db": "281e85f4-...",
    "top_k": 3
  }'
