Images

Generate images with Amazon Titan Image Generator v2, upload existing images with automatic AI vision analysis (Claude), and manage your image library. All images are indexed for RAG search via companion text files.

All image endpoints require authentication. Include Authorization: Bearer <idToken> in your request headers.
POST

Generate Image

/images/generate

Generate an image from a text prompt using Amazon Titan Image Generator v2. The generated image (1024x1024 PNG) is stored in S3 and indexed for search.

Request Body
ParameterTypeDescription
promptstring requiredText description of the image to generate.
knowledgeDbIdstring requiredTarget knowledge base ID.
Response Fields (201)
FieldTypeDescription
imageIdstringUnique image identifier.
promptstringOriginal prompt.
providerstringAlways "aws".
imagestringBase64-encoded PNG image data.
messagestringSuccess confirmation.
 
// POST /images/generate
// Header: Authorization: Bearer <idToken>
{
  "prompt": "A futuristic city skyline at sunset",
  "knowledgeDbId": "281e85f4-00b8-4613-b518-a3b4679e8d4c"
}
// 201 Created
{
  "imageId": "c3d4e5f6-a7b8-9012-cdef-123456789012",
  "prompt": "A futuristic city skyline at sunset",
  "provider": "aws",
  "knowledgeDbId": "281e85f4-...",
  "image": "iVBORw0KGgoAAAANSUhEUg...",
  "message": "Image generated and stored successfully"
}
POST

Upload Image

/images/upload

Upload an image file. Claude Vision automatically analyzes the image and generates a text description, which is stored as a companion file for RAG search indexing. Max size: 5MB. Supported: PNG, JPEG, GIF, WebP.

Form Fields
FieldTypeDescription
filefile requiredImage file to upload.
knowledgeDbIdstring requiredTarget knowledge base ID.
Response Fields (201)
FieldTypeDescription
imageIdstringUnique image identifier.
filenamestringOriginal filename.
descriptionstringAI-generated image description.
messagestringSuccess confirmation.
 
// POST /images/upload
// Content-Type: multipart/form-data
// Header: Authorization: Bearer <idToken>

file: <binary image data>
knowledgeDbId: "281e85f4-..."
// 201 Created
{
  "imageId": "d4e5f6a7-b8c9-0123-defg-234567890123",
  "filename": "architecture-diagram.png",
  "knowledgeDbId": "281e85f4-...",
  "description": "The image shows a system architecture diagram with microservices connected via an API gateway...",
  "message": "Image uploaded and processed successfully"
}
# cURL example
curl -X POST \
  https://fgbpqt2pq6.execute-api.us-east-1.amazonaws.com/prod/images/upload \
  -H "Authorization: Bearer <idToken>" \
  -F "file=@./diagram.png" \
  -F "knowledgeDbId=281e85f4-..."

GET

Get Image

/images/{imageId}

Retrieve metadata for a specific image, including a pre-signed S3 download URL valid for 1 hour.

Path Parameters
ParameterTypeDescription
imageIdstring requiredImage ID to retrieve.
// 200 OK
{
  "imageId": "c3d4e5f6-...",
  "prompt": "A futuristic city skyline",
  "provider": "aws",
  "fileSize": 524288,
  "createdAt": "2025-07-04T14:00:00.000000",
  "status": "generated",
  "downloadUrl": "https://s3.amazonaws.com/..."
}
DELETE

Delete Image

/images/{imageId}

Permanently delete an image. Removes the image file, companion text description, and all metadata sidecars from S3. Triggers a KB sync to clean up indexed data.

Path Parameters
ParameterTypeDescription
imageIdstring requiredImage ID to delete.
// 200 OK
{
  "message": "Image 'architecture-diagram.png' deleted successfully"
}
POST

Reindex (Regenerate) Image

/images/reindex

Regenerate a previously generated image using its original prompt. Re-uploads the new image to S3, updates the companion text file, and triggers a KB sync. Only works for AI-generated images (not uploaded ones).

Request Body
ParameterTypeDescription
imageIdstring requiredID of the generated image to regenerate.
Only images that were originally generated via POST /images/generate can be regenerated. Uploaded images will return a 400 error.
// POST /images/reindex
// Header: Authorization: Bearer <idToken>
{
  "imageId": "c3d4e5f6-a7b8-9012-cdef-123456789012"
}
// 200 OK
{
  "imageId": "c3d4e5f6-a7b8-9012-cdef-123456789012",
  "prompt": "A futuristic city skyline at sunset",
  "provider": "aws",
  "message": "Image regenerated successfully"
}
